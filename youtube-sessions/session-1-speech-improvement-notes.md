## Grammar Patterns to Watch

### 1. Article Usage (a/an/the)

Non-native speakers often omit or misuse articles:

| Example from Transcript | Improved Version |
|------------------------|------------------|
| "create web application" | "create **a** web application" |
| "Grado is powerful tool" | "Gradle is **a** powerful tool" |
| "This is account ID" | "This is **the** account ID" |

**Tip**: When introducing something for the first time, use "a/an". When referring back to it, use "the".

### 2. Preposition Usage

| Example | Improved |
|---------|----------|
| "deploy to production as first iteration" | "deploy to production **as the** first iteration" |
| "nice template" | (OK, but could add "a nice template") |

## Grammar Quick Reference

### Articles (a/an/the)

**Use "a/an" for:**
- First mention of something: "Let's create **a** new branch"
- One of many: "This is **a** great framework"

**Use "the" for:**
- Something specific: "**The** main branch is protected"
- Second mention: "Remember **the** branch we created?"

**No article for:**
- Plural generalizations: "Developers love Kotlin"
- Proper nouns: "GitHub offers..."

### Common Preposition Patterns

- "deploy **to** production"
- "push **to** remote"
- "merge **into** main"
- "work **on** a feature"
- "switch **to** a branch"
