## Grammar Patterns (Ongoing)

### Article Usage (a/an/the)

From both sessions, article omission is the most common pattern:

| Example | Improved Version | Session |
|---------|------------------|---------|
| "create web application" | "create **a** web application" | 1 |
| "it's important one" [03:17] | "it's **an** important one" | 2 |
| "from build configuration perspective" [05:36] | "from **a** build configuration perspective" | 2 |

### Preposition Patterns

| Pattern | Example |
|---------|---------|
| deploy **to** | "deploy to production" |
| migrate **to** | "migrate to the new plugin" |
| address **the** | "address the issue" |
| build **on** | "build on what we have" |
